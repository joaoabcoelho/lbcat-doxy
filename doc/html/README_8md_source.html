00013 Installation:
00014 -------------
00015 
00016 You need to have cmake installed on your machine (this is on the
00017 standard repositories and probably already installed on your machine).
00018 CMT is not used anymore.
00019 
00020 Just checkout a new working copy of the software, but requiring a
00021 specific branch which is called cmake.
00022 
00023 Go to the main directory and adapt the setup.sh file. You
00024 should have only to modify the CATPATH variable, so that it points to
00025 the directory where the software is installed.
00026 
00027 Type
00028 
00029 > source setup.sh
00030 
00031 > mkdir build; cd build
00032 
00033 > cmake ..
00034 
00035 > make all
00036 
00037 > make install
00038 The doxygen documentation can be built with the command
00039 
00040 > make doc
00041 
00042 Starting Cat:
00043 -------------
00044 
00045 To start cat you can run either in any place (alias defined in the
00046 setup.sh file):
00047 
00048 > catcmd
00049 for the command version
00050 
00051 > catgui
00052 for the gui version.
00053 It is advised to create another directory where you start the software,
00054 so that all the user files/directory will stand in another place than
00055 the source and compilation files.
00056 An alias setupCat is useful to launch the command
00057 
00058 > source setup.sh
00059 with the proper path to the source.sh file.
